# OpenChat
OpenChat is a Open-source Server and Client DIY Chat App. It can be a HomeLab Project.
## How To Setup?
> [!WARNING]
> This repo does not have any safety cautions, so anyone can join! Be aware of it
### Pre-setup
Before Running the Server and Client, Check if:
- The Server is connected Locally, not Globally (if connected, Open wide Door to get hacked.)
- Configure the Server Side and Client Side's Address and Port when compiling or in config.
### Server
The Server Can be run by either using source code(Compiling or running with py3).
### Client
The Client Side has two Types: TUI and GUI. You can choose it.
